Authentication Flow
User Fills Form
→
POST /api/auth/signup
→
bcrypt hashes password
→
Save to MongoDB
→
Return JWT
→
Store in localStorage
→
Redirect to Feed ✅
On every protected request → JWT sent in Authorization: Bearer <token> header → middleware verifies → attaches user to req

Database Schema (2 Collections)

👤 Users Collection

username String Unique handle required
email String Unique, lowercase required
password String bcrypt hashed required
avatar String URL or base64
bio String Max 150 chars
createdAt Date Auto timestamp

📝 Posts Collection

author ObjectId Ref → User required
username String Denormalized (fast reads)
text String Max 1000 chars
image String base64 or URL
likes ObjectId[] Array of user IDs
comments Array Embedded sub-docs

API Endpoints
REST API — Base URL: /api
Method Endpoint Auth Description
POST /auth/signup — Register new user
POST /auth/login — Login, receive JWT
GET /auth/me JWT ✅ Get current user profile
GET /posts?page=1&limit=10 — Public paginated feed
POST /posts JWT ✅ Create post (text/image)
PUT /posts/:id/like JWT ✅ Toggle like on post
POST /posts/:id/comment JWT ✅ Add comment to post
DELETE /posts/:id JWT ✅ Delete own post

Optimistic UI — How It Works

❤️ Like Toggle

1

User clicks Like

Heart fills immediately. Count increments in local state. Zero waiting.

2

API call fires in background

PUT /posts/:id/like sent asynchronously.

3

Sync with server

On success → replace with server's likes array. On failure → revert to original state.

💬 Comment

1

User submits comment

Comment appears instantly in the list with a temporary ID.

2

API call in background

POST /posts/:id/comment sent. Input clears immediately.

3

Replace temp with real

Server returns real comment → temp ID replaced. On failure → comment removed, text restored.

Quick Local Setup
bash — Terminal 
# 1. Clone repo
git clone https://github.com/yourusername/social-app.git
cd social-app

# 2. Backend setup
cd backend
npm install
cp .env.example .env   # then edit .env with your values
npm run dev            # → http://localhost:5000

# 3. Frontend setup (new terminal)
cd ../frontend
npm install
# Create frontend/.env:
echo "REACT_APP_API_URL=http://localhost:5000/api" > .env
npm start              # → http://localhost:3000 🎉
